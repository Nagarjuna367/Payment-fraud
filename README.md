# Payment-fraud
This project implements an optimized machine learning system for online payment fraud detection. It includes data analysis, feature engineering, model training, hyperparameter tuning, and deployment using Flask. The best-performing model is selected and served for real-time fraud prediction via a web application.

How to Access the Application :

Clone the repository to your local machine

  git clone <repository-url>

Navigate to the project directory

  cd Online-Payments-Fraud-Detection

Install required dependencies

  pip install -r requirements.txt

Run the Flask application

  python app.py

Open a web browser and go to

  http://127.0.0.1:5000/

Enter transaction details in the input form and submit to get a real-time fraud prediction result.

Access the output using these URL: http://localhost:5000/predict or http://192.168.55.103:5000
